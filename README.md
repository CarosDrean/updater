# UPDATER

Updater es un simple paquete para copiar directorios y sus archivos.

# Instalacion

```
go get
```

# Configuracion

Crear el archivo *configuration.json* a partir del *configuration.example.json*.

# Compilacion

Para compilar el proyecto use:

```
go build .
```
